# 🌦 Weather & Air Quality App

A modern weather application that provides real-time weather updates and air quality index (AQI) information based on the searched location.

---


## 📌 Features

- 🌍 Search weather by city name
- 🌡 Real-time temperature updates
- 💧 Humidity levels
- 🌬 Wind speed (km/h)
- 🌫 Air Quality Index with color indicators
- 📅 Dynamic date display
- 🌤 Weather icons from OpenWeatherMap
- 🎨 Modern UI design

---

## 🛠 Built With

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

---

## 📡 API Used

- Weather API
- Air Pollution API

Provided by OpenWeatherMap.

---
